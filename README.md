# Ecsctrl

This gem gives you cli control over your ECS clusters

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ecsctrl'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ecsctrl

## Usage
